This folder contains the hardware modules:

- `gen_chain` hardware module for accelerating WOTS-chain computations

- `gen_leaf` hardware module for accelerating XMSS-leaf computation

- `gen_pk` hardware module for generating the public key in WOTS 

- `l_tree` hardware module for the l_tree computation

- `seed_expansion` hardware module generating the secret key in WOTS

- `thash_f` hash computation F

- `thash_h` hash computation H
